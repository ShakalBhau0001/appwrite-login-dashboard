## Configuration

Replace the following placeholders in JavaScript files with your own Appwrite details:

- YOUR_APPWRITE_ENDPOINT
- YOUR_PROJECT_ID
- YOUR_DATABASE_ID
- YOUR_COLLECTION_ID
